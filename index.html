ST.JS Tutorial

W3.JS é uma biblioteca JavaScript projetada para simplificar projetos de desenvolvimento web:

Fácil de aprender e fácil de usar.

Projetado para desenvolvimento rápido de aplicativos.

Projetado para todos os dispositivos: PC, Tablet e Celular.

Grátis para usar. Sem licenças.

O que o ST.JS pode fazer?

Abaixo está uma lista de algumas das coisas que o ST.JS pode fazer:

  • Executar apresentação de slides
  • Incluir (importar) HTML em HTML
  • (Sorteia) - Classifique o conteúdo em qual quer documento HTML
  • Filtre qualquer elemento HTML
  • Exibe qualquer elemento JavaScript em HTML
  • Olculta e mostra qualquer elemento HTML
  • Adicionar e remove classe em qualquer elemento HTML
  • Adicionar CSS em qualquer elemento HTML

Como

ST.JS é muito fácil de usar.

Basta adicionar um link para st.js em sua página da web:

Exemplo:

<script src="st.js"></script>
